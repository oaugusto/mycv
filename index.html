MAJOR PROJECTS

CBNet – Counting-Based Self-Adjusting Tree Networks

A counting-based self-adjusting networks that scales with the amount of concurrency by using balance information in the nodes.

  • Designing an extension of DisplayNet[1] into a scalable concurrent self-adjusting network using counter.
  • Analysis of time and routing path per operation using amortized analysis.
  • Simulating in the Sinalgo platform to validate the algorithm (using Java in the framework).

DisplayNet – Distributed Self-Adjusting Networks

First distributed self-adjusting network fully decentralized that adapts dynamically to the communication pattern.

  • Designed and developed of the distributed algorithms.
  • Simulation in the Sinalgo platform to validate the algorithm (using Java in the framework).
  • The system was tested by using different scenario in concurrent communication requests from real Data Centers.
  • Published the paper [1] in INFOCOM 2019.

Matrix – Multihop Address Assignment for Any-to-Any Routing with 6LoWPAN

A platform-independent routing protocol that enables reliable and efficient any-any data traffic in 6LoWPAN.

  • Implemented the protocol in the simulation platform TinyOS using nesC (C-like dialect).
  • Simulation of several scenarios to validate the proposed protocol.
  • Published the paper [2] in an international conference for wireless networks.
